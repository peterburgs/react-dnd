<hr/>

## How to start project

1. Clone project and cd to the project's directory
2. Install all dependencies:

```bash
yarn
```

3. Start json-server:

```bash
yarn server
```

<strong>Make sure the file db.json is in form of:</strong>

```json
{
  "widgets": {}
}
```

4. Start React app in another terminal:

```bash
yarn dev
```
